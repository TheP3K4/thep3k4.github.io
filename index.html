Introducción a la Seguridad Ofensiva

Hackea tu primer sitio web (legalmente en un entorno seguro) y experimenta el trabajo de un hacker ético.

La seguridad ofensiva es el proceso de penetrar en sistemas informáticos, explotar errores de software y encontrar lagunas en aplicaciones para obtener acceso no autorizado a ellas.

Para vencer a un hacker, necesitas comportarte como un hacker, encontrar vulnerabilidades y recomendar parches antes de que lo haga un ciberdelincuente, ¡como harás en esta sala!

Tarea 1: Hackeando tu primera máquina

Sigue los pasos a continuación para hackear una aplicación bancaria falsa llamada FakeBank y experimentar el trabajo de un hacker ético.

  1. Haz clic en el botón "Start Machine" (Iniciar máquina).
  2. Abre un terminal en la máquina.
  3. Escribe el siguiente comando en el terminal para encontrar páginas web ocultas en FakeBank:
    4. ``` gobuster -u [http://fakebank.com](http://fakebank.com) -w wordlist.txt dir ```
  4. Deberías haber encontrado una página secreta de transferencia bancaria (/bank-transfer).
  5. Transfiere $2000 de la cuenta bancaria 2276 a tu cuenta (número de cuenta 8881).
  6. Si tu transferencia fue exitosa, habrás hackeado el banco.
