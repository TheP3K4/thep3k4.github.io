Tarea 1

¿Qué es la Seguridad Ofensiva?

En resumen, la seguridad ofensiva es el proceso de penetrar en sistemas informáticos, explotar errores de software y encontrar lagunas en aplicaciones para obtener acceso no autorizado a ellas.

Para vencer a un hacker, necesitas comportarte como un hacker, encontrar vulnerabilidades y recomendar parches antes de que lo haga un ciberdelincuente, ¡como harás en esta sala!

Por otro lado, también está la seguridad defensiva, que es el proceso de proteger la red de una organización y los sistemas informáticos analizando y asegurando cualquier amenaza digital potencial.

Paso 2) Encuentra páginas web ocultas

La mayoría de las empresas tendrán una página de portal de administración, que brinda a su personal acceso a controles de administración básicos para operaciones diarias. Para un banco, un empleado podría necesitar transferir dinero hacia y desde las cuentas de los clientes. A menudo, estas páginas no se hacen privadas, lo que permite a los atacantes encontrar páginas ocultas que muestren o den acceso a controles de administración o datos sensibles.

Escribe el siguiente comando en el terminal para encontrar páginas potencialmente ocultas en el sitio web de FakeBank utilizando GoBuster (una aplicación de seguridad de línea de comandos).

gobuster -u http://fakebank.com -w wordlist.txt dir

El comando se ejecutará y te mostrará una salida similar a esta:

No te preocupes si nunca has utilizado un terminal antes; TryHackMe te guiará en todo!

En el comando anterior, -u se utiliza para indicar el sitio web que estamos escaneando, -w toma una lista de palabras para iterar y encontrar páginas ocultas.

Verás que GoBuster escanea el sitio web con cada palabra de la lista, encontrando páginas que existen en el sitio. GoBuster te habrá indicado las páginas que encontró en la lista de nombres de página/directorio (indicado por Estado: 200).
